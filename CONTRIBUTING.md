# Contributing

このリポジトリは CLI 設定と運用メモを共有するためのドキュメント中心プロジェクトです。

## 変更の基本方針
- 1コミット1トピックで変更する
- 先に該当 OS / 共通ディレクトリへ追記し、ルート README からリンクする
- 既存の文体（簡潔・実行手順中心）に合わせる

## 推奨フロー
```bash
git checkout -b docs/<topic>
# 編集
git status
git diff -- README.md common/ macOS/ Ubuntu/ Windows/
```

## 記述ルール
- 見出しは `#` / `##` / `###` を段階的に使う
- コマンドは `bash` コードブロックで示す
- ユーザー固有情報（ユーザー名、秘密情報、絶対パス）は含めない

## 事前チェック
- 相対リンクが切れていない
- コマンド例がコピー実行可能
- 変更対象OSが明確に書かれている

## Pull Request
PR には次を含めてください。
- 目的（何を改善したか）
- 変更ファイル一覧
- 影響範囲（macOS / Ubuntu / Windows / common）
- 表示変更がある場合のみスクリーンショット
